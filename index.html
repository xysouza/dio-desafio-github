Títulos

As tags de títulos em HTML são usadas para definir a estrutura hierárquica de um documento HTML, ou seja, para criar títulos e subtítulos que organizam o conteúdo da página. As tags de títulos são representadas por h1, h2, h3, h4, h5 e h6, onde h1 é o título mais importante e h6 é o menos importante.

Body

A tag body é um elemento fundamental em um documento HTML. Ela representa o corpo principal do documento HTML e contém todo o conteúdo visível que será exibido no navegador. Isso inclui texto, imagens, links, elementos de mídia, formulários e praticamente todo o conteúdo que os visitantes do site verão ao acessar a página da web. É dentro da tag body que os desenvolvedores web organizam e estruturam o conteúdo para criar páginas da web atraentes e funcionais. Em resumo, a tag body é o "coração" do documento HTML, onde tudo o que os usuários veem e interagem é definido.

Parágrafos

A tag <p> em HTML é utilizada para definir parágrafos de texto. Ela é uma das tags mais básicas e amplamente usadas em HTML. Quando você coloca texto dentro de uma tag <p>, o navegador exibirá esse texto como um parágrafo separado, geralmente com espaço em branco acima e abaixo do texto para indicar a separação entre os parágrafos.

(Voltar)

Links

A tag <a>, que significa "âncora", é uma das tags mais importantes e amplamente utilizadas em HTML. Ela é usada para criar hiperlinks, permitindo que você vincule uma página da web a outra, uma seção de uma página a outra, ou até mesmo a recursos externos, como documentos, imagens ou sites.

(Voltar)

Listas

As tags <ul> e <ol> são usadas em HTML para criar listas. <ul> representa uma "Lista Não Ordenada" e <ol> representa uma "Lista Ordenada".
